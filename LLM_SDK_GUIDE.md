# Sybilion Go SDK — LLM / codegen single reference (`go.sybilion.dev/sybilion`)

Copy facts from this file; do not invent module paths, env names, or forecast shapes.

## Go version

**Go 1.25+** (see `go.mod` in this directory).

## Module path and dependency

- **Module:** `go.sybilion.dev/sybilion`
- **Wrapper package:** `package sybilion` at the module root (`sybilion.New`, `sybilion.Options`, `sybilion.AsGenericOpenAPIError`, ...)
- **Generated low-level client:** `package sybilionapi` at `go.sybilion.dev/sybilion/api`

Prefer a **versioned** dependency from the module proxy:

```bash
go get go.sybilion.dev/sybilion@v0.1.0
```

For local development without publishing, use **`replace`** in the app `go.mod`:

```go
require go.sybilion.dev/sybilion v0.0.0

replace go.sybilion.dev/sybilion => /absolute/path/to/sybilion-go
```

(See [`example/go.mod`](./example/go.mod).)

## Environment variables (client process)

| Variable | Required | Role |
|----------|----------|------|
| `SYBILION_API_TOKEN` | Yes for real calls | Bearer token: API key `sk_ops_...` or a dashboard session token. Read in your app and pass `sybilion.Options{Token: ...}`. The wrapper does **not** read this env var by itself. |
| `SYBILION_API_BASE_URL` | No | When `Options.BaseURL` is empty, `sybilion.New` uses this env var, then the compiled default `sybilion.DefaultPublicAPIBaseURL`. (`OPERATIONAL_API_BASE_URL` is accepted as a deprecated alias for one transition release.) |

## Default API origin

`DefaultPublicAPIBaseURL` is defined in **`defaults_gen.go`**, generated by **`go run ./cmd/embed-go-defaults`** from **`defaults.json`** at the repository root. Override at runtime with **`SYBILION_API_BASE_URL`** or **`Options.BaseURL`**.

## Minimal `GET /api/v1/me`

```go
package main

import (
	"context"
	"log"
	"os"

	"go.sybilion.dev/sybilion"
)

func main() {
	c := sybilion.New(sybilion.Options{
		Token: os.Getenv("SYBILION_API_TOKEN"),
	})
	me, _, err := c.DefaultAPI().ApiV1MeGet(context.Background()).Execute()
	if err != nil {
		log.Fatal(err)
	}
	log.Printf("user_id=%s balance_eur_cents=%d", me.GetUserId(), me.GetBalanceEurCents())
}
```

## Minimal async forecast + wait

Requirements from the API: **`frequency` must be `"monthly"`**; **`timeseries`** must have **at least 60** monthly points (YYYY-MM-DD keys).

```go
package main

import (
	"context"
	"log"
	"os"
	"time"

	"go.sybilion.dev/sybilion"
	api "go.sybilion.dev/sybilion/api"
)

func main() {
	ctx := context.Background()
	c := sybilion.New(sybilion.Options{
		Token: os.Getenv("SYBILION_API_TOKEN"),
	})

	meta := api.NewTimeseriesMetadata("Synthetic monthly demo series")
	ts := make(map[string]float32)
	anchor := time.Date(2021, 1, 1, 0, 0, 0, 0, time.UTC)
	for i := range 60 {
		d := anchor.AddDate(0, i, 0)
		ts[d.Format("2006-01-02")] = 100
	}

	req := api.NewForecastRequestV1("v1", 6, "monthly", 0.5, *meta, ts)
	acc, _, err := c.DefaultAPI().ApiV1ForecastsPost(ctx).ForecastRequestV1(*req).Execute()
	if err != nil {
		log.Fatal(err)
	}

	job, err := c.Forecasts().Wait(ctx, acc.GetJobId(), 2*time.Second)
	if err != nil {
		log.Fatal(err)
	}
	log.Printf("settled=%v status=%s", job.GetSettled(), job.GetStatus())
}
```

## Errors (`GenericOpenAPIError`)

The generated client returns `*sybilionapi.GenericOpenAPIError` for many failures. Prefer **`errors.As`** or the helper **`sybilion.AsGenericOpenAPIError(err)`** (see `errors.go`).

```go
if ge, ok := sybilion.AsGenericOpenAPIError(err); ok {
	log.Printf("openapi error: %s body=%s", ge.Error(), string(ge.Body()))
	return
}
log.Fatal(err)
```

## Other languages

The official Python SDK is published as [`sybilion`](https://pypi.org/project/sybilion/) on PyPI (`pip install sybilion`). For any other language, generate a client from the public OpenAPI spec at **<https://api.sybilion.dev/openapi.yaml>**.
