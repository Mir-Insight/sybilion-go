# Go SDK — LLM / codegen single reference (`github.com/Mir-Insight/developers-portal-api-sdk-go`)

Copy facts from this file; do not invent module paths, env names, or forecast shapes.

## Go version

**Go 1.25+** (see `go.mod` in this directory).

## Module path and dependency

- **Module:** `github.com/Mir-Insight/developers-portal-api-sdk-go`
- **Generated API/models package:** `github.com/Mir-Insight/developers-portal-api-sdk-go/gen` (Go package name `devportal`)
- **Hand-written wrapper:** import as `devportalclient` from `github.com/Mir-Insight/developers-portal-api-sdk-go`

Prefer a **versioned** dependency from the module proxy:

```bash
go get github.com/Mir-Insight/developers-portal-api-sdk-go@v0.1.0
```

For local development without publishing, use **`replace`** in the app `go.mod`:

```go
require github.com/Mir-Insight/developers-portal-api-sdk-go v0.0.0

replace github.com/Mir-Insight/developers-portal-api-sdk-go => /absolute/path/to/developers-portal-api-sdk-go
```

(See [`example/go.mod`](./example/go.mod).)

## Environment variables (client process)

| Variable | Required | Role |
|----------|----------|------|
| `OPERATIONAL_API_TOKEN` | Yes for real calls | Bearer token: API key `sk_ops_...` or Auth0 access token. Read in your app and pass `devportalclient.Options{Token: ...}`. The wrapper does **not** read this env var by itself. |
| `OPERATIONAL_API_BASE_URL` | No | When `Options.BaseURL` is empty, `devportalclient.New` uses this env var, then the compiled default `devportalclient.DefaultPublicAPIBaseURL`. |

Do **not** confuse with server/worker `OPS_API_BASE_URL` in the ops-api deployment `.env`.

## Default API origin

`DefaultPublicAPIBaseURL` is defined in **`defaults_gen.go`**, generated by **`go run ./cmd/embed-go-defaults`** from **`defaults.json`** at the repository root. Override at runtime with **`OPERATIONAL_API_BASE_URL`** or **`Options.BaseURL`**.

## Minimal `GET /api/v1/me`

```go
package main

import (
	"context"
	"log"
	"os"

	devportalclient "github.com/Mir-Insight/developers-portal-api-sdk-go"
)

func main() {
	c := devportalclient.New(devportalclient.Options{
		Token: os.Getenv("OPERATIONAL_API_TOKEN"),
	})
	me, _, err := c.DefaultAPI().ApiV1MeGet(context.Background()).Execute()
	if err != nil {
		log.Fatal(err)
	}
	log.Printf("user_id=%s balance_eur_cents=%d", me.GetUserId(), me.GetBalanceEurCents())
}
```

## Minimal async forecast + wait

Requirements from the API: **`frequency` must be `"monthly"`**; **`timeseries`** must have **at least 60** monthly points (YYYY-MM-DD keys).

```go
package main

import (
	"context"
	"log"
	"os"
	"time"

	devportalclient "github.com/Mir-Insight/developers-portal-api-sdk-go"
	devportal "github.com/Mir-Insight/developers-portal-api-sdk-go/gen"
)

func main() {
	ctx := context.Background()
	c := devportalclient.New(devportalclient.Options{
		Token: os.Getenv("OPERATIONAL_API_TOKEN"),
	})

	meta := devportal.NewTimeseriesMetadata("Synthetic monthly demo series")
	ts := make(map[string]float32)
	anchor := time.Date(2021, 1, 1, 0, 0, 0, 0, time.UTC)
	for i := range 60 {
		d := anchor.AddDate(0, i, 0)
		ts[d.Format("2006-01-02")] = 100
	}

	req := devportal.NewForecastRequestV1("v1", 6, "monthly", 0.5, *meta, ts)
	acc, _, err := c.DefaultAPI().ApiV1ForecastsPost(ctx).ForecastRequestV1(*req).Execute()
	if err != nil {
		log.Fatal(err)
	}

	job, err := c.Forecasts().Wait(ctx, acc.GetJobId(), 2*time.Second)
	if err != nil {
		log.Fatal(err)
	}
	log.Printf("settled=%v status=%s", job.GetSettled(), job.GetStatus())
}
```

## Errors (`GenericOpenAPIError`)

The generated client returns `*devportal.GenericOpenAPIError` for many failures. Prefer **`errors.As`** or the helper **`devportalclient.AsGenericOpenAPIError(err)`** (see `errors.go`).

```go
if ge, ok := devportalclient.AsGenericOpenAPIError(err); ok {
	log.Printf("openapi error: %s body=%s", ge.Error(), string(ge.Body()))
	return
}
log.Fatal(err)
```

## Other languages

This repository ships a **first-party Go wrapper** plus Python and TypeScript helpers under `sdks/`. For any other language, generate a client from the OpenAPI document shipped with the Go generated tree: **`gen/api/openapi.yaml`** (copy of the wire contract at generation time), or the canonical spec in the monorepo at `internal/httpapi/openapi/openapi.yaml`.
